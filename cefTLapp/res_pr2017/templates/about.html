About Translate 2017

Translate 2017 is an automated translation service specially designed for the 2017 Estonian Presidency of the Council of the European Union.

The service provides the world’s best automated translation systems for Estonian, built with the latest innovations in language technology. The systems can be used to translate texts, full documents, and websites.

Translate 2017 also integrates the European Commission’s CEF eTranslation platform, which offers automated translation systems for translating between the 24 official languages of the EU.

The Estonian-language automated translation systems and user interface of Translate 2017 were developed by Tilde, a language technology company based in Tallinn, Riga, and Vilnius.
